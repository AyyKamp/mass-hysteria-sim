# Mass Hysteria Simulator

Simulation utility for the [Hearthstone card Mass Hysteria][mh].

[Demo here.][demo]

Simulation code originally done in R by [TelegramSam98][reddit] and ported to JS by [ChocoShell][gh]

## Project setup

```npm install```

### Test locally

```npm run serve```
